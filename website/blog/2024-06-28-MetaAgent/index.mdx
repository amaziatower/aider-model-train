**TL;DR**
- We introduce MetaAgent, an agent equipped with the capability to dynamically assemble a group of agents to handle complex tasks via NestedChat.
- MetaAgent is supported by a comprehensive agent library as well as a tool library. The agents will go through retrieval-selection-generation process before put into NestedChat. They will be equipped with different tools as well.
